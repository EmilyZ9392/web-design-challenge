Weather Analysis: Impact of Latitude on Various Weather Conditions


Using data from OpenWeatherMap API on weather conditions for over 500 cities, we were able to analyze how distance from the equator affects various weather conditions.

Once we had this data, we used Matplotlib to visualize our findings for Cloudiness, Temperature, Humidity, and Wind Speed.

This site houses all of the data used to conduct our analysis, as well as the visualizations created with the data. It also provides explanations of the trends/correlations that we observed.
